## 구성품

 * 라즈베리파이4 4g
 * nema17 40mm 스텝모터 x/y 모터 각1개
 * nema17 펜케익모터 익스트루더 1개
 * 핫엔드 - 멜로우 v6핫엔드
 * bmg익스트루더(속 내용물만 사용)
 * 옴론 금속센서(프로브 오토레벨링을 위해 사용)
 * 메인보드 - skr mini e3 v2.0
